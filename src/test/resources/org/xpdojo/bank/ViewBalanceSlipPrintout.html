Viewing a Balance Slip Printout

Once we're able to relay a customer's current balance, we really should be able to present it in a way that gets all the information across and looks as you'd expect.

The difficultly is in describing "as you'd expect". That's where this test comes in.

Example

Jane has an account with us.

Jane checks her balance.

Jane should see a pretty balance slip.
